# Genomics-Project
Automated Bash pipeline for variant analysis in family trio. From raw FASTQ to VCFs. Supports Mendelian inheritance models (AR, de novo, AD) using Bowtie2, FreeBayes, Bedtools and BCFtools. Includes integrated QC reports (FastQC, Qualimap, MultiQC).
